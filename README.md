# Private Equity Deal Intelligence Dashboard

A Python and Streamlit dashboard for screening acquisition targets and running a lightweight mini LBO analysis.

## Overview
This project simulates the top of a private equity sourcing and underwriting funnel. Users can filter companies by growth, profitability, valuation, and leverage, then evaluate return potential through a simplified LBO framework.

## Features
- Interactive company screening
- Ranking logic for investment targets
- Mini LBO modeling
- MOIC and IRR outputs
- Exit multiple and growth sensitivity
- Downloadable underwriting output

## Tech Stack
- Python
- Pandas
- Streamlit

## How to Run
```bash
pip install -r requirements.txt
python -m streamlit run app.py
